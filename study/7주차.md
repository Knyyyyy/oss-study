# 7주차 OSS 정리

- **SemVer방식**
  - major.minor.patch
  - 세 자리 숫자 형태로 표기하는 버전
  - **메이저(major) 번호**
    - 첫 자리가 0으로 시작하면 아직 초기 개발 중인 제품이라는 의미
    - 정식 버전은 1부터 시작
  - **마이너(minor) 번호**
    - 메이저 버전에서 기능을 추가하거나 변경 사항이 있을 때 바꿈
  - **패치(patch) 번호**
    - 버그 수정 등 미미한 변화가 있을 때 바꿈
- **태그 생성**
  - $ git tag v1.0.1
- **주석태그 생성**
  - $ git tag –a v1.0.1 –m ‘msg’
- **태그 자세한 정보 확인**
  - $ git show v1.0.1
- **태그 목록 확인**
  - $ git tag
- **태그 삭제**
  - $ git tag –d v1.0.1
- **저장소 목록 보기**
  - $ git branch
- **저장소 생성만**
  - $ git branch <new-branch>
- **저장소 생성하고 이동**
  - $ git checkout -b <new-branch>
  - git switch –c <new-branch>
- **저장소 삭제**
  - $ git branch –d branch-name
- **저장소 삭제, 강제 삭제**
  - $ git branch –D branch-name
