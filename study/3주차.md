# 3주차 OSS 정리

- **깃설정 명령 구조**
  - $git config — 설정범위 설정변수 설정값
- **설정범위**
  - —system : 모든사용자에게 적용
  - —global: 현재 사용자의 모든 저장소에 적용
  - —local: 현재 사용자의 현재 저장소에만 적용
- **global인경우 설정한 내용은 .gitconfig에 저장**
- **설정 명령(예시들어설명)**
  - 사용자이름:$git config —global user.name"ai7dnn"
  - 사용자 전자 메일: $git config —global user.email ai7dnn@gmail.com
  - 줄바꿈 자동전환: $git config —global core.autocrlf true
  - 줄바꿈 안전설정: $git config —global core.autocrlf false
  - 기본 편집기 설정: $git config —global core.editor 'code —wait'
  - 기본 브랜치 이름: $git config —global init.defaultBranch main
- **깃 저장소 생성**
  - $ git init = $git init . ->현재 디렉토리를 깃 저장소로 생성
  - $git init baisc: 현재 폴더 basic을 생성하고 깃 저장소로 만듬
- **리눅스 명령**
  - $pwd: 현재 폴더 표시
  - $cd:디렉토리 이동
  - $mkdir dname:디렉토리 생성
  - $ls:목록 조회
    - l: 파일의 상세정보
    - a: 숨김파일 표시
    - al: 상세정보와 숨김파일 표시
  - $touch: 빈 파일 생성
  - $echo: 문자열을 컴퓨터 터미널에 출력
  - $cat: 파일의 내용을 화면에 출력
  - $cp a b: 파일 a를 b로 복사
  - $mv f1 f2:파일 f1을 f2로 이름 수정
  - $rm:파일 삭제
    - f: 강제로 삭제
    - r:디렉토리 내부의 모든 내용을 삭제
- **Redirection 명령어 연산**
  - >:기존 파일 내용을 지우고 저장
  - >>: 기존 파일 내용 뒤에 덧붙여서 저장
  - <: 파일 데이터를 명령에 입력
